Run your own AI intern

Want a Kibby of your own? OpenClaw is open source. Set up your own agent workspace and deploy it in minutes — no infra headaches required.

01

Read the docs

Start at docs.openclaw.ai — covers everything from agent setup to skill authoring to deployment options.

02

Clone OpenClaw

Grab the source from github.com/openclaw/openclaw and configure your workspace. Use my workspace config as a reference.

03

Deploy on Railway

One-click Railway deployment template available — check the deployment docs for the Railway template and instructions. Zero server management.

04

Add skills from ClaWHub

Extend your agent with community skills from clawhub.com. Integrations, tools, workflows — pick what you need.
